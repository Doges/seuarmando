#SEU ARMANDO

já sabe né

## Compatibility

- Linux
- Windows
- Mac

## Installing
Install globally:

    npm install -g seuarmando

## Running
Run command:

    seuarmando

Or import in NodeJS script:

    import seuarmando from 'seuarmando';

    seuarmando();
    
## Running with [npx](https://www.npmjs.com/package/npx)
Run command:

    npx seuarmando
